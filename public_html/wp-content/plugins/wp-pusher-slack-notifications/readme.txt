Get a notification in Slack every time something is deployed from Git with WP Pusher.

== Description ==

WP Pusher lets you install and update plugins and themes directly from GitHub and Bitbucket. This extension will notify you in Slack every time something has been deployed.

Whenever you push something to your Git repository on GitHub or Bitbucket, and it has been deployed to WordPress with WP Pusher, you will be pinged in Slack to know that everything went OK.

<h4>NB: Requires WP Pusher</h4>

<p><a href="https://wppusher.com/#pricing">Download WP Pusher here</a></p>
</blockquote>

== Installation ==

You can install WP Pusher Slack Notifications by searching for it on WordPress.org or you can use WP Pusher to install it diretly from <a href="https://github.com/wppusher/wppusher-slack">GitHub</a>.
